# 💪 PROGRESSAR

> App de progressão de carga inteligente baseado no **Método Paulo Muzy**

---

## 🚀 Como usar

Acesse diretamente pelo GitHub Pages ou clone e abra o `index.html` no navegador.

**Para instalar como app (PWA):**
1. Abra no Chrome/Safari
2. Toque em "Adicionar à tela inicial"
3. Use como app nativo

---

## ✨ Funcionalidades

- 📈 **Progressão automática** de carga baseada no Método Muzy
- 🎯 **Modo Foco** — interface minimalista durante o treino
- ⏱ **Timer de descanso** com anel SVG animado
- 🎵 **Cadência** com beeps e síntese de voz (pt-BR)
- 🧬 **Biblioteca ADN** — 12 grupos musculares detalhados
- 🏆 **Gamificação** — streak, XP, badges
- 📊 **Histórico** com gráficos
- ⚠️ **Alertas inteligentes** de platô e fadiga
- 💾 **Modo offline** — dados salvos localmente

## 📁 Estrutura do repositório

```
/
├── index.html          # App completo (single file)
├── manifest.json       # Configuração PWA
├── service-worker.js   # Cache offline

└── icons/
    ├── icon-192.svg    # Ícone PWA 192px
    └── icon-512.svg    # Ícone PWA 512px
```

## 🛠 Tecnologias
